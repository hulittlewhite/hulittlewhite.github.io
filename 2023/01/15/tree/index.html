🌟 二叉树算法题思考总结：

  1. 是否可以通过遍历一遍二叉树得到答案？如果可以，用一个 traverse 函数配合外部变量来实现。
  2. 是否可以定义一个递归函数，通过子问题（子树）的答案推导出原问题的答案？如果可以，写出这个递归函数的定义，并充分利用这个函数的返回值。
  3. 无论使用哪一种思维模式，都要明白二叉树的每一个节点需要做什么，需要在什么时候（前中后序）做。

Reference: labuladong 的算法小抄

104. 二叉树的最大深度

题源：104. 二叉树的最大深度

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

Solution

Solution 1: 动态规划

  • 输入一个节点，返回以该节点为根的二叉树的最大深度。
  • 根据左、右子树的最大深度得到原二叉树的最大深度。

Solution 2: 回溯

  • 遍历二叉树，在遍历的过程中记录最大深度。

Code

Code 1: 动态规划

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) return 0;
        int leftMax = maxDepth(root.left);
        int rightMax = maxDepth(root.right);
        return 1 + Math.max(leftMax, rightMax);
    }
}

Code 2: 回溯

  • 新增类的成员变量：
    • res 用于记录最大深度；
    • depth 用于记录当前递归到的节点深度。
  • 新增方法 traverse() ：遍历二叉树。
    • 前序位置是进入一个节点的时候，在前序位置 depth 自增 1；
    • 后序位置是离开一个节点的时候，在后序位置 depth 自减 1。
class Solution {
    int depth = 0;
    int res = 0;

    public int maxDepth(TreeNode root) {
        traverse(root);
        return res;
    }

    void traverse(TreeNode root) {
        if (root == null) return;
        depth++;
        res = Math.max(res, depth);
        traverse(root.left);
        traverse(root.right);
        depth--;
    }
}

144. 二叉树的前序遍历

题源：144. 二叉树的前序遍历

给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

Solution

Solution 1: 动态规划

  • 输入一个节点，返回以该节点为根的二叉树的前序遍历结果。
  • 前序遍历结果特点：第一个是根节点的值，接着是左子树，最后是右子树。

Solution 2: 回溯

  • 新增方法 traverse() ：遍历二叉树，先遍历根节点，再遍历左子树，后遍历右子树。

Code

Code 1: 动态规划

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        LinkedList<Integer> res = new LinkedList<>();
        if (root == null) return res;
        res.add(root.val);
        res.addAll(preorderTraversal(root.left));
        res.addAll(preorderTraversal(root.right));
        return res;
    }
}

Code 2: 回溯

class Solution {
    LinkedList<Integer> res = new LinkedList<>();

    public List<Integer> preorderTraversal(TreeNode root) {
        traverse(root);
        return res;
    }

    void traverse(TreeNode root) {
        if (root == null) return;
        res.add(root.val);
        traverse(root.left);
        traverse(root.right);
    }
}

94. 二叉树的中序遍历

题源：94. 二叉树的中序遍历

给定一个二叉树的根节点 root ，返回 它的 中序 遍历 。

Solution

  • 回溯
    • 新增方法 traverse() ：遍历二叉树，先遍历左子树，再遍历根节点，后遍历右子树。

Code

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    LinkedList<Integer> res = new LinkedList<>();
  
    public List<Integer> inorderTraversal(TreeNode root) {
        traverse(root);
        return res;
    }

    void traverse(TreeNode root) {
        if (root == null) return;
        traverse(root.left);
        res.add(root.val);
        traverse(root.right);
    }
}

145. 二叉树的后序遍历

题源：145. 二叉树的后序遍历

给定一个二叉树的根节点 root ，返回 它的 后序 遍历 。

Solution

  • 回溯
    • 新增方法 traverse() ：遍历二叉树，先遍历左子树，再遍历右子树，后遍历根结点。

Code

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    LinkedList<Integer> res = new LinkedList<>();
  
    public List<Integer> postorderTraversal(TreeNode root) {
        traverse(root);
        return res;
    }

    void traverse(TreeNode root) {
        if (root == null) return;
        traverse(root.left);
        traverse(root.right);
          	res.add(root.val);
    }
}

102. 二叉树的层序遍历

题源：102. 二叉树的层序遍历

给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

Solution

  • 采用队列，先将根节点入队。
  • 从上到下遍历二叉树的每一层：
    • 从左到右遍历每一层的每个节点。

Code

  • while 循环控制从上到下一层层进行遍历。
  • for 循环控制每一层从左到右进行遍历。
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new LinkedList<>();
        if (root == null) return res;

        Queue<TreeNode> q = new LinkedList<>();
        q.offer(root);

        while (!q.isEmpty()) {
            int q_length = q.size();
            List<Integer> level = new LinkedList<>();
            for (int i = 0; i < q_length; i++) {
                TreeNode curr = q.poll();
                level.add(curr.val);
                if (curr.left != null) {
                    q.offer(curr.left);
                }
                if (curr.right != null) {
                    q.offer(curr.right);
                }
            }
            res.add(level);
        }
        return res;
    }
}

剑指 Offer II 045. 二叉树最底层最左边的值

题源：剑指 Offer II 045. 二叉树最底层最左边的值

给定一个二叉树的 根节点 root，请找出该二叉树的 最底层 最左边 节点的值。

假设二叉树中至少有一个节点。

Solution

  • 新增两个类的成员变量
    • maxDepth 用于记录二叉树的最底层所在层数。
    • res 用于记录结果节点的值。

Code

  • 新增 traverse() 方法：
    • 根据遍历的当前节点所在层，选择是否更新 maxDepth。
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    int maxDepth = 0;
    int res = 0;
    
    public int findBottomLeftValue(TreeNode root) {
        traverse(root, 1);
        return res;
    }

    void traverse(TreeNode root, int depth) {
        if (root == null) return;
        if (depth > maxDepth) {
            maxDepth = depth;
            res = root.val;
        }
        traverse(root.left, depth + 1);
        traverse(root.right, depth + 1);
    }
}
